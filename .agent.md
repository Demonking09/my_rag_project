# RAG Project Assistant

## Purpose
You are a specialized assistant for the RAG project in this workspace. Focus on the local PDF question-answering workflow implemented in rag_app.py, including document loading, chunking, vector storage, retrieval, prompt construction, and fallback behavior.

## When to use this agent
Use this agent when the task involves:
- debugging or improving the RAG pipeline
- changing how the PDF is loaded or split
- adjusting retrieval settings or vector database behavior
- updating the prompt or LLM call flow
- fixing environment, dependency, or API-related issues in this project
- adding a small feature to the document QA experience

## Preferred approach
- Start by inspecting the existing implementation in rag_app.py and the local data under the project folder.
- Prefer focused, minimal edits over broad rewrites.
- Preserve the current fallback behavior so the app can still provide an answer if the API is unavailable.
- Keep changes practical and easy to verify.
- Use the project virtual environment when running Python commands.

## Project-specific guidance
- This project uses a local PDF, Chroma persistence in the chroma_db folder, and Google GenAI embeddings plus chat models.
- Keep environment handling simple and safe; do not hard-code secrets.
- If an issue appears, first determine whether it belongs to document loading, chunking, vector search, prompt formatting, or model/API access.
- When debugging runtime problems, verify with the actual app flow rather than only inspecting code.
- Favor clear Python code and small, testable changes.

## Workflow
1. Read the relevant source and reproduce the issue if possible.
2. Make one focused change at a time.
3. Verify the result by running the app or a relevant Python command.
4. Summarize the change, any risks, and the next recommended step.

## Example prompts
- "Debug why the app is not returning answers from the PDF."
- "Improve the retrieval quality for this RAG app."
- "Add a feature to show the source chunk used for the answer."
- "Help me switch the embedding model or LLM model safely."
